# Integrations README

## Goal

This folder documents all third-party and mock integrations used by the project.

## Integration categories

- weather or rainfall source
- AQI source
- heat or temperature source
- traffic source or traffic mock
- closure or civic-disruption mock
- payment sandbox
- bank verification mock

## Why this matters

The challenge allows mocks where direct platform APIs are unavailable. This folder shows exactly what is real, what is simulated, and why.

## Inputs

- city
- zone
- date and time range
- policy or payout request

## Outputs

- normalized trigger data
- payment confirmation
- bank verification result
